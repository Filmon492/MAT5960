\section{Numerical Experiments}
In the previous section we have devolved the Lax--Friedrichs type scheme and the Godunov type Scheme for the nonlocal model. We described in particular the complete algorithm on how to implement the Lax--Friedrichs type scheme. In this section we consider if the numerical solutions converge to the correct solution numerically.
\subsection{Comparison of Left endpoint  and Normalized Left endpoint }
We start by comparing the quadrature weights obtained from the left endpoint  and the normalized left endpoint .
For the Riemann problem \eqref{eq: Riemann problem}, we choose $u_L = 0.6$ and $u_R = 0.1$ such that the problem becomes to solve
\begin{equation}\label{eq:sloving rarefaction}
    \begin{split}
        u_{t} + f(u)_{x} & = 0 \\
        u(x,0)           & =
        \begin{cases}
            0.6 & \text{if } x < 0 \\
            0.1 & \text{if } x > 0
        \end{cases}
    \end{split}
\end{equation}
Since $f(u) = u(1-u)$ is a convex function, $h$ in \eqref{eq:h} equals $f$ itself. Thus, the entropy solution is given by:
\begin{equation}\label{eq: u_L > u_R}
    u(x,t) =
    \begin{cases}
        0.6               & \text{for } x \leq -0.2t,           \\
        \dfrac{t - x}{2t} & \text{for } -0.2t \leq x \leq 0.8t, \\
        0.1               & \text{for } x \geq 0.8t.
    \end{cases}
\end{equation}

We plot $u$ at final time $ T = 1$ over the domain $[x_L, x_R] = [-2, 2]$ together with both the local and nonlocal numerical approximations. The linear decreasing kernel $\omega_{\epsilon}(y) = \frac{2(\epsilon-y)}{\epsilon^2}$ is used and the numerical viscosity constant $\alpha$ is set to $2$.\\

We begin by implementing the Lax--Friedrichs type scheme for $\epsilon = 0.1$ and $\Delta x = 0.02$

Finally taking $\epsilon = 0.001$ and $\Delta x = 0.004$,  the numerical solution of the nonlocal model obtained using the left endpoint numerical quadrature weights completely fails. By contrast, significantly better accuracy is observed when using the normalized left endpoint numerical quadrature weights.

Summary.\\
The numerical experiments above demonstrate that, as $\Delta x$ and $\epsilon$ tend to zero simultaneously, while the Lax--Friedrichs type scheme based on the left endpoint quadrature weights becomes increasingly inaccurate and eventually fails to produce a meaningful numerical solution, the normalized left endpoint quadrature weights consistently maintain stability and improved alignment with the entropy solution. These results indicate that the choice of numerical quadrature weights plays a crucial role in ensuring consistency between the Lax--Friedrichs scheme and the local model. The same applies for the Godunov type scheme as well, it turns out that for the sake of consistency it's required that the numerical quadrature weights must satisfy the normalized condition $\sum_{k=0}^{m-1} w_k = 1$.

\subsection{Numerical Convergence Analysis}
Based on the numerical experiments and arguments we provided above, both the Lax--Friedrichs scheme and Godunov scheme are  consistent and stable when using the normalized left endpoint quadrature weights.
Let us continue the numerical experiment to determine the convergence and order of convergence, and see if one scheme is more accurate than the other.

Denote:
\begin{itemize}
    \item $U^{0}$: the reference solution of the local model.
    \item $U^{0, \Delta x}$: the numerical solution of the local model.
    \item $U^{\epsilon}$: the reference solution of the nonlocal model.
    \item $U^{\epsilon, \Delta x}$: the numerical solution of the nonlocal model.
\end{itemize}
We are interested in presenting numerical experiments on whether:
\begin{itemize}
    \item for fixed $\epsilon$, $U^{\epsilon, \Delta x} \rightarrow U^{\epsilon} $ as $\Delta x \rightarrow 0$.
    \item $U^{\epsilon, \Delta x} \rightarrow U^{0} $ as both $\Delta x \rightarrow 0$ and $\epsilon \rightarrow 0$ simultaneously.
\end{itemize}


Considering the computed reference and numerical solutions as piecewise constant functions on each cell of size $\Delta x$, we measure the size of the global error $e = U^{ref} - U^{num}$ using the $L^{1}$ norm:
\begin{align*}
    \| e \Vert_{L^{1}} = \Delta x \sum_{j=0}^{K-1} | e^j \vert.
\end{align*}
where $U^{ref}$ and $U^{num}$ corresponding to the the reference and numerical solution respectively. We desire to get $\| e \Vert_{L^{1}} \rightarrow 0 $ as the parameters tend to zero.

\subsection{Numerical Convergence to the Nonlocal Model}

We begin investigating if the devolved numerical schemes converge numerically to the solution of the nonlocal model. For fixed $\epsilon$, through numerical experiments we aim to deduce $\| U^{\epsilon} - U^{\epsilon, \Delta x}\Vert_{L^{1}}= \| e \Vert_{L^{1}} \rightarrow 0$ as $\Delta x \rightarrow 0$.

Since the analytic solution is not always available, it is practical to compute the reference solution numerically on a very fine grid. Focusing on the initial data given in \eqref{eq:sloving rarefaction}, we compute the reference solution $U^{\epsilon}$ for $\Delta x = \frac{x_R-x_L}{4096} = \frac{4}{4096}$ and $\epsilon = 0.1$. Holding $\epsilon = 0.1$ fixed and letting $\Delta x \rightarrow 0$, although $\Delta x = \frac{1}{2^{4+m}}$ for $m = 0,1,2,3, 4, 5, 6$, we compute the $L^{1}$ errors and plot against the number of cells to compare the efficiency of the Godunov scheme and Lax--Friedrichs scheme. We also preset the observed order of convergence in Table~\ref{tab:table 1}
\begin{figure}[H]
    \centering
    \includegraphics[width=0.7\textwidth]{Conver_2l.pdf}
    \caption{$L^1$ error vs. number of cells}
    \label{fig:Conver_1}
\end{figure}
Figure~\ref{fig:Conver_1} indicates that the $L^{1}$-error for both Godunov and Lax--Friedrichs scheme tends to zero as the number of cells increases.
In other words, for fixed $\epsilon$ both the schemes converge to the reference (weak) solution $U^{\epsilon}$ of the nonlocal model as $\Delta x \to 0$. We observe as well that the Godunov scheme converges faster than Lax--Friedrichs scheme. The reason leis on how the schemes are devolved, The Godunov scheme doesn't involve the numerical viscosity constant $\alpha$ at all, while the Lax--Friedrichs scheme does, which results in less efficiency.
A natural question is: how fast do the schemes converge?
To answer this, we study the order of convergence $r$ for both schemes.

We assume that the $L^{1}$ error on a mesh of size $\Delta x_i$ can be written as
\begin{align*}
    E_i = C(\Delta x_i)^r,
\end{align*}
for some constant $C>0$.
If we have the error at two consecutive refinement levels $(E_{i-1},E_i)$, then
\begin{align*}
    \frac{E_{i-1}}{E_i}
    =\left(\frac{\Delta x_{i-1}}{\Delta x_i}\right)^{r}.
\end{align*}

Taking logarithms, we obtain the standard expression for the order of convergence $r$:
\begin{align*}
    r =
    \frac{\log(E_{i-1}) - \log(E_i)}
    {\log(\Delta x_{i-1}) - \log(\Delta x_i)}.
\end{align*}
The order of convergence tells us how fast does a scheme converge to the correct solution, the higher order is the faster convergence.
