## Project Summary
This project simulates nonlocal traffic flow model using numerical methods inspired by finite volume methods. The programing languge is Python and the implementation is object-oriented and uses vectorized code to ensure efficiency and scalability. The code can be found under the folder src. 

This project is being conducted under the supervision of Professor Ulrik Skre Fjordholm at the University for Oslo (UiO).
