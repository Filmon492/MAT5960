# MAT5960 — Master Project

## Author
Filmon Misgane

## Project Summary
This project simulates a nonlocal traffic flow model using numerical methods inspired by finite volume methods. The programming language is Python, and the implementation is object-oriented and uses vectorized code to ensure efficiency and scalability. The code can be found in the folder "src".

This project was conducted under the supervision of Professor Ulrik Skre Fjordholm at the University of Oslo (UiO).
